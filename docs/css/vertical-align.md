# vertical-align [참고자료](https://blogpack.tistory.com/773)
- 테이블 태그의 셀 태그인 <td>에 사용가능
- 인라인 속성을 가진 요소에 사용가능, line-height 높이 범위 안에서 수직 정렬
- 참고로 테이블 셀에 적용하는 속성 값은 top, middle, bottom만 적용 가능, 나머지 속성은 인라인 요소에 적용가능
