# cursor
요소 위에 마우스 커서가 올라갔을 때 보여줄 모양을 지정합니다.

``` css
cursor: default; //보통 화살표.
cursor: none; //커서 없음.
cursor: help; //도움말 사용 가능.
cursor: pointer; // 링크를 나타내는 포인터. 보통 한 쪽을 가리키는 손.
cursor: progress; //프로그램이 백그라운드에서 작업 중이지만, 사용자가 인터페이스와 상호작용할 수 있음. (wait과 반대)
```

등등.. 여러가지가 존재합니다.
